How To Use

To navigate between the pages, please refer to the navigation tabs at the top of the page. From the home page, there will be options to either go view / add finances, see more about us, or to see this guide. You will also be provided the option to log out.


font-family:"courier">The Finance tab is where all finances will be managed. You will immediately see two drop-down menus, in which you can view subcodes for a specific set of gender and sport. There will also be the option to add new purchases with options such as the date, amount, notes, and more provided. Upon adding new fiancial information, you will need to confirm if you really want this added or not.
